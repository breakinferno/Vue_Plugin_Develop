# lfment

> A Vue.js project. To complete a comment 'system'

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 技术栈

Vue + MongoDB + Koa2 + RxJS
